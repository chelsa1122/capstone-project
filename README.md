# capstone-project
## First time Setup

### 1. Install npm libs

```
npm install
```

### 2. Setup database tables

```
npm run migrate
```


### 3. Add fake data

```
npm run seed
```

### Run ui server 
```
npm run dev 
```
### Run Backend
- Create database
```
npm run migrate
```
- Run api server

```
npm run start
```
